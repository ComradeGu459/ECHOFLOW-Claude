# Project Roadmap: English Learning Studio

## Phase 1: AI-Powered Content Ingestion (Current Focus)
- [ ] **YouTube/Local Video Parsing:** Implement real video downloading or local file handling.
- [ ] **AI Transcription:** Integrate whisper-v3 or similar API for high-accuracy transcription.
- [ ] **Bilingual Alignment:** Use LLM (Gemini/GPT) to align L1 and L2 subtitles.
- [ ] **Expression Extraction:** AI-driven identification of key idioms, collocations, and grammar points.

## Phase 2: Core Studio Experience
- [ ] **Pronunciation Feedback:** Integrate Azure Speech or Google Speech-to-Text for real-time phoneme-level feedback.
- [ ] **AI Explanation (Contextual):** Deep-link Studio expressions to LLM for contextual usage examples.
- [ ] **Shadowing Mode:** Auto-pause and repeat loops for sentence-by-sentence practice.

## Phase 3: Spaced Repetition (SRS) & Analytics
- [ ] **Anki-style SRS:** Implement Anki-like scheduling (SM-2 algorithm) for flashcards.
- [ ] **Detailed Analytics:** Track phoneme-level pronunciation trends over time.
- [ ] **Learning Path:** AI-generated learning paths based on vocabulary gaps.

## Phase 4: Personalization & Export
- [ ] **Custom Vocab Lists:** Export flashcards to Anki/Quizlet.
- [ ] **Personal AI Tutor:** Voice-based conversational practice based on imported video topics.
